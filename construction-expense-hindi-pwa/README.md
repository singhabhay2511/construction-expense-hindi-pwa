# Construction Expense Hindi PWA

A very simple Hindi-first expense app for civil construction work.

## Features

- Add projects
- Open a project
- View summary
- Add one expense item at a time
- Enter amount
- Local browser storage
- Hindi UI
- Basic Hindi voice input support where browser allows

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy on Vercel, Netlify, or GitHub Pages.

## iPhone use

Open the deployed URL in Safari and tap **Add to Home Screen**.
